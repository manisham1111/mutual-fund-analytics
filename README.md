# Mutual Fund Analytics Capstone Project

## Project Overview

This project analyzes mutual fund industry data using Python, SQLite, and Power BI. The objective is to generate insights on fund performance, investor behavior, SIP trends, and risk metrics through data analytics and interactive dashboards.

## Tools Used

* Python
* Pandas
* NumPy
* SQLite
* Power BI
* GitHub

## Datasets Used

1. Fund Master
2. NAV History
3. AUM by Fund House
4. Monthly SIP Inflows
5. Category Inflows
6. Industry Folio Counts
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices

## ETL Workflow

Raw CSV Data
→ Data Cleaning & Transformation
→ SQLite Database
→ Analytics Processing
→ Power BI Dashboard
→ Business Insights

## Dashboard Pages

### Page 1: Industry Overview

## How to Run

1. Clone the repository.
2. Install required packages:
   pip install -r requirements.txt
3. Run ETL scripts.
4. Execute notebooks for analytics.
5. Open Power BI dashboard (.pbix file).

## Key Findings

* SIP inflows increased consistently.
* Equity funds attracted higher investments.
* Investors aged 26–35 contributed the most investments.
* High Sharpe Ratio funds showed better risk-adjusted returns.
* VaR and CVaR effectively measured downside risk.
